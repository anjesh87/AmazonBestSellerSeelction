Project Title: Adding Best seller headphones to the cart.

Motivation: Edkal Job Challenge #1

Code Style: Java 8 (Using Lambda expression)

Framework Used: Page Object Model

Technology Used : Java, TestNG, Selenium, Maven

Features: Automated the adding best seller headphones from first search page.

Installation: Need to have Java 8

How to use:

#1. Import the project by pom.xml or by unzipping the folder and do manual import
#2. Use FireFox latest (recommended).
#3. Either it can run by selecting HomePageTest class or by testNG xml file.
